## 🔧 구현 내용

### 1️⃣ AdminNav 컴포넌트 (관리자 페이지)

**파일:** `/components/admin/AdminNav.tsx`

#### 기능
- 데스크톱(md 이상): 가로 메뉴 표시
- 모바일(md 미만): 햄버거 메뉴 버튼 표시
- 클릭 시 드롭다운 메뉴 표시/숨김

#### 메뉴 항목
```
1. 메인 페이지 (Home 아이콘)
2. 대시보드 (Dashboard 아이콘)
3. 크루즈 관리 (Package 아이콘)
4. 설정 (Settings 아이콘)
```

#### 코드 구조
```typescript
// 상태 관리
const [mobileMenuOpen, setMobileMenuOpen] = useState(false);

// 메뉴 항목 배열
const menuItems = [
  { href: "/", label: "메인 페이지", icon: Home },
  { href: "/dashboard/partner", label: "대시보드", icon: LayoutDashboard },
  { href: "/admin/cruises", label: "크루즈 관리", icon: Package },
  { href: "/admin/settings", label: "설정", icon: Settings },
];

// 해당 항목을 매핑하여 메뉴 생성
```

#### 스타일링
```tailwindcss
/* 데스크톱 메뉴 */
hidden md:flex

/* 모바일 버튼 */
md:hidden

/* 드롭다운 */
md:hidden mb-4 bg-white rounded-lg border
```

### 2️⃣ 메인 페이지 (app/page.tsx)

**파일:** `/app/page.tsx`

#### 구현 변경사항

1. **임포트 추가**
   ```typescript
   import { Menu, X } from "lucide-react";
   ```

2. **상태 추가**
   ```typescript
   const [mobileMenuOpen, setMobileMenuOpen] = useState(false);
   ```

3. **모바일 버튼**
   ```jsx
   <button
     className="md:hidden text-white hover:text-[#FFD700]"
     onClick={() => setMobileMenuOpen(!mobileMenuOpen)}
   >
     {mobileMenuOpen ? <X className="w-6 h-6" /> : <Menu className="w-6 h-6" />}
   </button>
   ```

4. **메뉴 드롭다운**
   ```jsx
   {mobileMenuOpen && (
     <div className="md:hidden bg-[#004080] text-white border-b-2 border-[#FFD700]">
       <nav className="container mx-auto px-4 py-4 space-y-3">
         {/* 메뉴 항목들 */}
       </nav>
     </div>
   )}
   ```

#### 메뉴 항목
```
1. 크루즈
2. 패키지 검색 (Plane 아이콘)
3. 출발일정
4. 목적지
5. 회사소개
6. 관리자 (Settings 아이콘)
```

#### 디자인
- 배경색: `#004080` (짙은 파란색)
- 테두리: `#FFD700` (금색, 2px)
- 호버: `#003366` (더 짙은 파란색)
- 아이콘: lucide-react (Menu, X, Plane, Settings 등)

## 🔄 상호작용

### 클릭 시 동작
1. ☰ 버튼 클릭 → `mobileMenuOpen` = true
2. 메뉴 드롭다운 표시
3. 메뉴 항목 클릭 → 페이지 이동 + `mobileMenuOpen` = false (자동 닫힘)
4. ☰ 버튼 클릭 (열린 상태) → `mobileMenuOpen` = false
5. X 아이콘으로 변경

### 각 메뉴 항목 클릭 시
```javascript
onClick={() => setMobileMenuOpen(false)}
```
